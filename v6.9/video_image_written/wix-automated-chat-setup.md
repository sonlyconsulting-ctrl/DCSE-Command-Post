# Setting Up Automated Wix Chat

1. Add Wix Chat to Your Site:
   - In the Wix Editor, click "Add" (+)
   - Search for "Chat" and add it to your site

2. Set Up Automated Responses:
   - Go to your Dashboard
   - Click on "Inbox" in the left sidebar
   - Select "Automations" at the top

3. Create a Welcome Message:
   - Click "Add Automation" > "Welcome Message"
   - Customize the message (e.g., "Welcome to Sonly Consulting! How can we help you today?")

4. Set Up Quick Replies:
   - In the Welcome Message, add buttons for common queries (e.g., "Services", "Pricing", "Book a Consultation")

5. Create Topic-Based Automations:
   - For each topic, click "Add Automation" > "Topic-based"
   - Set a trigger keyword (e.g., "Services")
   - Create a response with information about your services

6. Design Conversation Flows:
   - Use Wix's visual flow builder to create more complex conversation paths
   - Include decision points where users can choose different options

7. Add AI-Powered Responses:
   - Enable Wix's AI Chat feature to handle more complex queries
   - Train it with your FAQs and common customer questions

8. Set Up Lead Capture:
   - Create an automation to ask for the user's email if they're interested in a consultation

9. Implement Handoff to Human:
   - Set up an option for users to request to speak with a real person
   - Ensure you have notifications set up to respond promptly

10. Test Your Chat:
    - Use preview mode to test your chat flows
    - Ensure all automations are working as expected

11. Monitor and Refine:
    - Regularly review chat transcripts to identify areas for improvement
    - Update your automations based on common questions or issues

Remember to keep your automated responses friendly and on-brand. Also, make it clear when they're chatting with a bot vs. a real person.
