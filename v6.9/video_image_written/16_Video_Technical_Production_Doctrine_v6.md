# VIDEO TECHNICAL PRODUCTION DOCTRINE

## Purpose
Define execution standards for video production.

## Specs
- Resolution: 4K–8K
- Frame Rate: 24fps (cinematic), 60fps (modern)
- Codec: ProRes / H.265
- Aspect: 16:9 / 2.35:1

## Pipeline
1. Script → prompt
2. AI generation
3. Edit (PowerDirector or equivalent)
4. Audio sync
5. Export

## Tools
- Veo / Kling (generation)
- AudioLM (voice)
- Editor (finalization)

## Rules
- Do not mix narrative content
- Maintain consistency across segments
- Ensure final render quality
